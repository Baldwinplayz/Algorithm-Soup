Number Sort

The default sorting function in javascript will be used to sort a numerical array.

Word Sort

This algorithm will sort words seperated by a whitespace.

Sentence Sort

This algorithm will sort sentences seperated by a newline using unicode numbers.

Sentence Length Sort

This algorithm will sort sentences seperated by a newlines based on how long they are, it can be word count, character count, etc.

Citation Sort

This algorithm will sort citation that are seperatred by a new line. This may require some editing to the citations.
