Description

This is a linear sorting algorithm; in other words, this algorithm operates on O(N) time. It will iterate through every item in the array to look for matches; all of these items are treated as strings, which means that you will not get matches for numerically equal numbers if they are written differently. This algorithm can be used to count the frequency of certain words in a text, although a histogram would be better for this.
